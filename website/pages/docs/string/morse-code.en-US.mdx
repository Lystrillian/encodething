## About Morse Code

Morse Code is a system of communication that uses a combination of dots and dashes to represent letters, numbers, and other symbols. It was developed by Samuel Morse in the mid-19th century and is still widely used today.

Morse Code is a visual representation of text that uses a combination of dots and dashes to represent letters, numbers, and other symbols. It was developed by Samuel Morse in the mid-19th century and is still widely used today.

## Usage

Morse Code is a utility function that can be used to convert text to Morse Code. Here's an example of how to use it:

```
INPUT: Hello, world!

OUTPUT: .... . .-.. .-.. --- / .-- --- .-. .-.. -.. -.-.--
```

## Parameters (Variant)

### Variant

The `variant` parameter can be used to specify the language of Morse Code. The default value is `english`.

```
INPUT: Hello, world!

OUTPUT: .... . .-.. .-.. --- / .-- --- .-. .-.. -.. -.-.--
```

```
INPUT: Привет, мир!

OUTPUT: .--. .-. .. .-- . - .-.-.- / -- .. .-. --..--
```

### Language Support

| Variant | Language |
| --- | --- |
| `english` | English (Latin) |
| `russian` | Russian (Cyrillic) |
| `japanese` | Japanese (Wabun) |
