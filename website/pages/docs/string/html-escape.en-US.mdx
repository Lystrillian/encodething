## About HTML escape

HTML Escape is the process of converting special characters in HTML into their corresponding HTML entities. This is useful when you need to display text in a web page that contains special characters.

HTML entities are represented as `&entity;` or `&#decimal;` or `&#hex;`. The entity can be a named entity (e.g., `&amp;`) or a numeric entity (e.g., `&#38;`).

HTML Escape is necessary because some characters have special meanings in HTML. For example, `<` and `>` are used to define HTML tags. If these characters appear in the text, they can cause unexpected behavior.
