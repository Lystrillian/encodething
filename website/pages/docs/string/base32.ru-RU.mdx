## О Base32

Base32 - это метод кодирования, использующий печатные символы ASCII.

В Base32 данные делятся на 5 бит и преобразуются в буквенно-цифровые символы (A-Z, 2-7). Преобразование выполняется через каждые 8 символов, а если последний символ меньше 8, то вставляется символ равенства (=).

Таблица преобразования для символов Base32 выглядит следующим образом.

| 5-битные данные | Base32 символ | 
| --- | --- |
| 00000 | A |
| 00001 | B |
| 00010 | C |
| 00011 | D |
| 00100 | E |
| 00101 | F |
| 00110 | G |
| 00111 | H |
| 01000 | I |
| 01001 | J |
| 01010 | K |
| 01011 | L |
| 01100 | M |
| 01101 | N |
| 01110 | O |
| 01111 | P |
| 10000 | Q |
| 10001 | R |
| 10010 | S |
| 10011 | T |
| 10100 | U |
| 10101 | V |
| 10110 | W |
| 10111 | X |
| 11000 | Y |
| 11001 | Z |
| 11010 | 2 |
| 11011 | 3 |
| 11100 | 4 |
| 11101 | 5 |
| 11110 | 6 |
| 11111 | 7 |

Например, если преобразовать слово "Hello!" в Base32, то оно будет выглядеть следующим образом.

1\. Представьте данные в бинарном представлении.

```
01001000 01100101 01101100 01101100 01101111 00100001  (For US-ASCII / UTF-8)
```

2\. Отделяйте каждые 5 бит. Если он меньше 5 бит, добавьте в конце "0"..

```
01001 00001 10010 10110 11000 11011 00011 01111 00100 00100
```

3\. Преобразуйте в символы с помощью таблицы преобразования. Преобразуйте каждые 8 символов, а если символов меньше 8, поставьте в конце знак "=".

```
JBSWY3DP EE======
```

4\. Соедините все символы, чтобы получить результат преобразования Base32.

```
JBSWY3DPEE======
```
