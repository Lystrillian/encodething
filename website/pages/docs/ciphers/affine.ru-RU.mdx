## О шифре Affine

Шифр Affine - это один из единственных транслитерационных шифров, который шифрует путем замены символов в тексте на другие символы.

Замена символов осуществляется по следующей формуле.

```
E(<var>x</var>) = (<var>a</var><var>x</var> + <var>b</var>) mod <var>m</var>
```

m - тип преобразуемого символа. x - буква, заменяемая числом от 0 до m - 1. a и bb - ключи шифрования.

Например, для 26 буквенных символов "ABCDEFGHIJKLMNOPQRSTUVWXYZ" m = 26, а x - от 0 до 25, причем от A до Z заменены цифрами.

Если a = 5, b = 3, то шифрование будет происходить в следующем потоке.

Результат шифрования всех букв с a = 5, b = 3 выглядит следующим образом.

```
Простой текст : ABCDEFGHIJKLMNOPQRSTUVWXYZ
Шифр: DINSXCHMRWBGLQVAFKPUZEJOTY
```

Указание a = 1, b = -3 дает шифр Цезаря, указание a = 1, b = 13 дает тот же результат, что и ROT13, а указание a = -1, b = -1 дает тот же результат, что и шифр Atbash.

a должно задавать число, со-кратное m. Это означает, что единственное число, которое делит a и m, должно быть 1. Например, a = 4, m = 26 не может быть задано, потому что оно делится как на 2, так и на 1. В Encodething, если задано недопустимое a , оно возвращает исходный символ без преобразования.
