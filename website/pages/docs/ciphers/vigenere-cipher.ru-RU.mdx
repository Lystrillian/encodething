### О шифре Виженера

Шифр Виженера - один из единственных транслитерационных шифров, который шифрует путем замены символов в тексте на другие символы.

Замена символов осуществляется с помощью ключа шифрования и следующего квадрата Виженера.

Верхняя строка - открытый текст до шифрования, левый столбец - ключ шифрования, а пересекающиеся символы - символы после шифрования. Ключ шифрования - это строка от 'A' до 'Z'.

Например, если открытым текстом до шифрования является "MESSAGE", а ключом шифрования - "SECRET", то символы заменяются и шифруются следующим образом. Поскольку первая буква - "M", а ключ - "S", зашифрованная буква будет "E".

```
Открытый текст : MESSAGE
Ключ: SECRETS
Шифрованный текст: EIUJEZW
```

Один символ ключа шифрования используется для каждого символа открытого текста. Если длина ключа короче длины открытого текста, используйте ключ многократно.
