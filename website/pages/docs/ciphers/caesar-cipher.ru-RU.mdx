## О шифре Цезаря

Шифр Цезаря - один из единых транслитерационных шифров, который шифрует путем замены символов в тексте на другие.

Замена символов осуществляется путем сдвига символов от "A" до "Z" среди 26 символов "ABCDEFGHIJKLMNOPQRSTUVWXYZ".

Например, при сдвиге -3 символа "A" зашифровывается в "X", а "Z" - в "W".

```
Простой текст : ABCDEFGHIJKLMNOPQRSTUVWXYZ
Шифр: XYZABCDEFGHIJKLMNOPQRSTUVW
```

```
Простой текст: ЭТО СЕКРЕТНОЕ СООБЩЕНИЕ
Шифрованный текст: QEFP FP X PBZOBQ JBPPXDB
```

Количество сдвигов является ключом к шифрованию.

Шифруются только буквы, но не цифры или символы.

Если число сдвигов равно 13, то результат будет таким же, как у ROT13. 

Сдвиг символов с сохранением диакритического знака. Так, например, буква "Á" зашифровывается в "X́".

## Поддержка других языков

Помимо латиницы, поддерживаются кириллица и японская хирагана/катакана.

#### Кириллица

Если вы хотите сдвинуть кириллический символ на -3 символа, он будет зашифрован следующим образом.

```
Простой : АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧЩЩЪЫЫЬЭЮЯ
Шифр: ЭЮЯАБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧЩЩЪЫЫЬ
```

Диакритический знак смещает символ при его удержании. Так, например, русская буква "Ё" зашифровывается в "В̈". При этом символы "Й" и "й" воспринимаются как уникальные, а не символы "И" и "и" с диакритическим знаком " ̆" (Бреве).
