## О Base45

Base45/Zlib/COSE/CBOR - это данные формата CBOR, подписанные в формате COSE, сжатые в формате Zlib и преобразованные в формат Base45.

[EUDCC (EU Digital COVID Certificate)](https://ec.europa.eu/info/live-work-travel-eu/coronavirus-response/safe-covid-19-vaccines-europeans/eu-digital-covid-certificate_en) используется в качестве формата данных для QR-кодов. EUDCC - это общий формат для сертификата вакцинации COVID-19 в ЕС, также известный как DGC (EU Digital Green Certificate) или Green Pass.

Encodething поддерживает только декодирование, а результат декодирования выражается в формате JSON. Действительность подписи не проверяется.

## О Base45

Base45 - это схема кодирования, использующая 7-битные печатаемые символы ASCII.

Base45 делит данные на два байта каждый и преобразует их в три ASCII-символа для представления.

Символы ASCII, используемые в Base45, выглядят следующим образом: рассматривайте 2-байтовое значение как беззнаковое целое число с большим порядком следования, вычисляйте каждый разряд (3 цифры) в 45-й десятичной системе, а затем вычисляйте результат преобразования Base45 на основе следующих символов ASCII.

Например, если преобразовать слово "Hello" в Base45, оно будет выглядеть следующим образом.

1\. Разделяйте каждые 2 байта.

```
4865<sub>(16)</sub> 6C6C<sub>(16)</sub> 6F<sub>(16)</sub>  (He ll o)
```

2\. Каждые 2 байта рассматриваются как беззнаковое целое число с большим порядком следования, и значение преобразуется в 3 цифры в каждой из 45 десятичных систем. Если конец - один байт, он преобразуется в 2 цифры в 45 десятичной системе.

```
4865<sub>(16)</sub>
= 18533<sub>(10)</sub>
= <b>9</b> * 45<sup>2</sup> + <b>6</b> * 45 + <b>38</b>
```

```
6C6C<sub>(16)</sub>
= 27756<sub>(10)</sub>
= <b>13</b> * 45<sup>2</sup> + <b>31</b> * 45 + <b>36</b>
```

```
6F<sub>(16)</sub>
= 111<sub>(10)</sub>
= <b>2</b> * 45 + <b>21</b>
```

3\. Преобразуйте каждые 3 цифры 45-й десятичной системы в символы ASCII в обратном порядке.

<table class="table table-sm table-fit"><tbody><tr><th>45 decimal digits</th><td>38</td><td>6</td><td>9</td><td></td><td>36</td><td>31</td><td>13</td><td></td><td>21</td><td>2</td></tr><tr><th>Base45 ASCII characters</th><td>%</td><td>6</td><td>9</td><td></td><td>[SP]</td><td>V</td><td>D</td><td></td><td>L</td><td>2</td></tr></tbody></table>

4\. Все символы соединяются, образуя результат преобразования Base45.

```
%69 VDL2
```
