## О шифре Atbash

Шифр Atbash - один из единственных транслитерационных шифров, который шифрует путем замены символов в тексте на другие.

Замена символов осуществляется путем отображения списка символов в обратном порядке.

Например, в алфавите "ABCDEFGHIJKLMNOPQRSTUVWXYZ" "A" зашифровывается в "Z", а "B" - в "Y".

```
Простой текст : ABCDEFGHIJKLMNOPQRSTUVWXYZ
Шифр: ZYXWVUTSRQPONMLKJIHGFEDCBA
```

```
Простой текст : THIS IS A SECRET MESSAGE
Шифрованный текст: GSRH RH Z HVXIVG NVHHZTV
```

Изначально он использовался как ивритский шифр. Ивритский шифр выглядит следующим образом.

```
Простой текст: אבגדהוזחטיכלמנסעפצקרשת
Шифррованный текст: תשרקצפעסנמלכיטחזוהדגבא
```

Поскольку существует взаимность: открытый текст может быть получен путем повторного шифрования шифротекста, дешифрование может быть выполнено в том же потоке, что и шифрование.

## Поддержка других языков

Помимо латиницы и иврита, поддерживаются кириллица и японская хирагана/катакана.


#### Кириллица

```
Простой : АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ
Шифрованный: ЯЮЭЬЫЪЩШЧЦХФУТСРПОНМЛКЙИЗЖЕДГВБА
```
